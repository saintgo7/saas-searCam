## 1. 수익 구조 개요

### 1.1 핵심 원칙

> "안전은 유료가 아닙니다" -- 모든 탐지 기능은 100% 무료

SearCam의 수익 모델은 **핵심 가치(안전 점검)를 무료로 유지**하면서, 편의 기능과 고급 기능에 대해 프리미엄을 부과하는 구조이다.

### 1.2 수익원 구조

```
SearCam 수익 구조
├── 1차 수익원: 프리미엄 구독 (Phase 1)
│   ├── 무제한 리포트 저장
│   ├── PDF 리포트 내보내기
│   ├── 광고 제거
│   ├── 우선 OUI DB 업데이트
│   └── 고급 통계/트렌드
│
├── 2차 수익원: 광고 (Phase 1)
│   ├── 홈 하단 배너
│   └── 결과 페이지 간 삽입 광고
│
├── 3차 수익원: B2B (Phase 2+)
│   ├── 숙박업소 인증 프로그램
│   ├── 기업용 대량 라이선스
│   └── API 라이선스
│
└── 4차 수익원: 부가 서비스 (Phase 3)
    ├── 전문 업체 연결 수수료
    └── 커뮤니티 프리미엄 기능
```

## 2. 무료 vs 프리미엄 기능 상세

### 2.1 기능 비교 표

| 기능 | 무료 | 프리미엄 |
|------|------|---------|
| Quick Scan (Wi-Fi 스캔) | O | O |
| Full Scan (3레이어 통합) | O | O |
| 렌즈 찾기 (Retroreflection) | O | O |
| IR Only 모드 | O | O |
| EMF Only 모드 | O | O |
| 교차 검증 + 위험도 산출 | O | O |
| 체크리스트 (숙소/화장실) | O | O |
| 결과 상세 보기 (근거 설명) | O | O |
| 112 신고 연동 | O | O |
| 리포트 저장 | **최근 10건** | **무제한** |
| PDF 리포트 내보내기 | X | O |

| 스캔 히스토리 비교 | X | O |
| 우선 지원 | X | O |

### 2.2 프리미엄 기능 상세 정의

#### 무제한 리포트 저장

- **무료**: 최근 10건만 저장. 11번째 저장 시 가장 오래된 리포트 자동 삭제 경고.
- **프리미엄**: 제한 없이 모든 리포트 영구 저장. 장소/날짜/위험도별 필터링.
- **전환 트리거**: 11번째 리포트 저장 시도 시 프리미엄 안내 팝업.

#### PDF 리포트 내보내기

- **기본 PDF**: 스캔 결과 요약, 발견 기기 목록, 위험도 등급, 일시/장소(선택).
- **커스텀 브랜딩**: PDF 상단에 사용자명 또는 업체명 삽입 (B2B 용도).
- **포함 내용**:
  - 스캔 일시 및 소요 시간
  - 각 레이어별 결과 상세
  - 교차 검증 결과 및 보정 계수
  - 발견된 모든 기기 목록
  - 위험도 등급 및 권장 조치
  - 면책 조항

#### 광고 제거

- **무료 사용자 광고 노출 위치**: 홈 화면 하단 배너, 결과 화면 이후 간 삽입 광고.
- **원칙**: 스캔 진행 중에는 절대 광고를 표시하지 않는다 (사용자 경험 보호).
- **프리미엄**: 모든 광고 완전 제거.

#### 우선 OUI DB 업데이트

- **무료**: 앱 업데이트(Play Store) 시에만 OUI 데이터베이스 갱신.
- **프리미엄**: 앱 실행 시 자동으로 최신 OUI DB 다운로드 (OTA). 신규 카메라 제조사를 빠르게 반영.
- **업데이트 주기**: 프리미엄 사용자는 주 1회, 무료 사용자는 앱 업데이트 주기(월 1~2회).

#### 고급 통계/트렌드

- **장소별 스캔 히스토리**: 같은 장소(Wi-Fi SSID 기반, 해시 처리)에서의 위험도 변화 추이.
- **개인 스캔 통계**: 총 스캔 수, 평균 위험도, 가장 많이 사용한 모드.
- **스캔 비교**: 두 개의 리포트를 나란히 비교하여 차이점 확인.

3. **연간 구독 강조**: 결제 화면에서 연간 구독을 기본 선택으로 표시

5. **학생 할인**: (Phase 2) 대학생 인증 시 40% 할인

### 4.4 eCPM 추정

| 광고 형태 | 예상 eCPM (한국) | 근거 |
|---------|----------------|------|
| 배너 | $1.0~$2.0 | 한국 평균 $1.5 |
| 간 삽입 | $5.0~$10.0 | 한국 평균 $7.0 |
| 네이티브 | $2.0~$4.0 | 한국 평균 $3.0 |

**일일 광고 수익 추정 (DAU 2,000 기준)**:
```
배너: 2,000 x 3 노출 x $1.5/1000 = $9.0/일
간 삽입: 2,000 x 0.5회 x $7.0/1000 = $7.0/일
합계: $16.0/일 = 약 ₩21,000/일 = 약 ₩630,000/월
```
