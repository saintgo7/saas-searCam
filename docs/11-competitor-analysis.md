## 2. 앱별 상세 분석

### 2.1 Hidden Camera Detector (FutureApps)

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 개발사 | FutureApps (인도) |
| 다운로드 | 10M+ (Play Store 기준) |
| 평점 | 3.6/5.0 (Play Store), 3.2/5.0 (App Store) |
| 마지막 업데이트 | 2025-11 |
| 앱 크기 | 45MB |
| 요금제 | 무료 + 프리미엄 (월 $4.99 ~ $9.99) |

#### 기능 목록

| 기능 | 설명 | 유/무료 |
|------|------|---------|
| AI 이미지 감지 | 카메라로 사물 촬영 → AI가 카메라 유무 판별 | 무료 (1일 3회) |
| Wi-Fi 스캔 | 같은 네트워크 기기 탐색 | 무료 |
| IR 감지 | 전면 카메라로 IR 발광 감지 | 무료 |
| EMF 감지 | 자기장 센서 활용 | 프리미엄 |
| 블루투스 스캔 | BLE 기기 탐색 | 프리미엄 |
| 리포트 | 스캔 결과 저장 | 프리미엄 |

#### UI 화면 구성 설명

- 홈: 중앙에 큰 "Scan" 버튼, 하단에 개별 모드 아이콘 나열
- 스캔 결과: 원형 게이지 + 숫자, 위험 기기 빨간 강조
- 탐지 모드: 각 모드별 별도 화면, 수동 전환 필요
- 설정: 감도 슬라이더, 구독 관리

#### 탐지 방식 분석

| 방식 | 구현 수준 | 정확도 추정 |
|------|---------|-----------|
| AI 이미지 | 서버 전송 후 분석 (온라인 필수) | 낮음 (30~40%). 일반 사물을 카메라로 오인 빈번 |
| Wi-Fi | ARP + 기본 포트 스캔 | 중간 (60~70%). OUI DB 부실 |
| IR | 기본 밝기 필터 | 중간 (50~60%). 어두운 환경 필수 |
| EMF | 단순 임계값 비교 | 낮음 (30~40%). 캘리브레이션 없음 |

#### 평점/리뷰 분석

| 카테고리 | 긍정 리뷰 패턴 | 부정 리뷰 패턴 |
|---------|-------------|-------------|
| 정확도 | "몇 번 제대로 찾아냄" (10%) | "모든 게 위험으로 뜸" (45%) |
| UI | "사용법이 직관적" (15%) | "광고가 너무 많다" (30%) |
| 요금 | - | "핵심 기능이 유료" (25%) |
| 기타 | "없는 것보다 나음" (20%) | "배터리 소모 심함" (15%) |

#### 핵심 약점

1. **오탐률 극심**: AI 이미지 분석이 일반 사물(볼트, 나사, 반사면)을 카메라로 오인
2. **핵심 기능 페이월**: EMF 감지가 프리미엄, AI 감지가 1일 3회 제한
3. **교차 검증 부재**: 각 센서 결과를 독립적으로 표시, 종합 판단 없음
4. **한계 미고지**: "위험" 표시만 하고, 왜 위험한지 근거 없음
5. **광고 과다**: 스캔 중에도 전면 광고 표시

#### 기본 정보

| 항목 | 내용 |
|------|------|

| 다운로드 | 1M+ |

| 마지막 업데이트 | 2026-01 |
| 앱 크기 | 32MB |
| 요금제 | 무료 + 프리미엄 (연 $19.99) |

#### 기능 목록

| 기능 | 설명 | 유/무료 |
|------|------|---------|
| Wi-Fi 스캔 | ARP + mDNS | 무료 |
| 블루투스 스캔 | BLE 기기 탐색 | 무료 |
| IR 감지 | 전면 카메라 IR | 무료 |
| EMF 감지 | 자기장 + 실시간 그래프 | 무료 |
| 렌즈 감지 | 기본 플래시 반사 (단순) | 프리미엄 |
| 통합 스캔 | 순차 스캔 | 프리미엄 |

#### UI 화면 구성 설명

- 홈: 5개 모드 아이콘 그리드, 개별 진입
- 각 모드: 전용 UI (Wi-Fi = 기기 목록, EMF = 그래프, IR = 카메라 뷰)
- 결과: 모드별 개별 결과 화면, 통합 결과 없음 (프리미엄만 통합)
- 디자인: Material Design 2 스타일, 다소 구식

#### 탐지 방식 분석

| 방식 | 구현 수준 | 정확도 추정 |
|------|---------|-----------|
| Wi-Fi | ARP + mDNS, OUI DB 보통 | 중상 (70~75%) |
| BT | BLE 스캔 + 알려진 카메라 필터 | 중간 (50~60%) |
| IR | 밝기 필터 + 지속 시간 분석 | 중상 (65~75%, 암실) |
| EMF | 캘리브레이션 + 실시간 표시 | 중간 (45~55%) |
| 렌즈 | 단순 밝기 임계값 (retroreflection 분석 없음) | 낮음 (35~45%) |

#### 핵심 약점

1. **수동 모드 전환**: 5개 모드를 사용자가 하나씩 전환해야 함
2. **통합 결과 유료**: 무료 사용자는 각 모드 결과를 개별적으로만 확인
3. **렌즈 감지 미흡**: retroreflection 패턴 분석 없이 단순 밝기만 체크
4. **결과 설명 부족**: "의심 기기 발견" 정도만 표시
5. **한국어 미지원**: 영어만 지원

## 4. SearCam 차별화 포인트 상세 분석

### 4.1 교차 검증 (최대 차별점)

기존 앱은 각 센서 결과를 독립적으로 보여줍니다. SearCam은 3개 레이어의 결과를 가중치 기반으로 교차 검증하여 종합 위험도를 산출합니다.

| 항목 | 기존 앱 | SearCam |
|------|--------|---------|
| 결과 표시 | 센서별 독립 결과 | 종합 위험도 (0~100) |
| 자기장 단독 양성 | "위험!" 표시 | 보정 x0.5 + "확인 필요" |
| Wi-Fi + IR 동시 양성 | 각각 "위험" 표시 | 보정 x1.4 + 구체적 근거 |
| 오탐률 | 55~70% | 목표 5~10% |

### 4.2 플래시 Retroreflection 렌즈 감지 (기술 차별점)

기존 앱의 렌즈 감지는 단순 밝기 임계값만 적용합니다. SearCam은 retroreflection의 물리적 특성(포인트 크기, 원형도, 위치 안정성, 플래시 의존성)을 AI로 분석합니다.

| 항목 | 기존 앱 | SearCam |
|------|--------|---------|
| 분석 방식 | 밝기 > 임계값 | 6단계 AI 파이프라인 |
| 오탐 필터 | 없음 | 크기/형태/위치/시간축/동적 검증 |
| 밝은 환경 | 사용 불가 | 사용 가능 (핵심 장점) |
| 전원 OFF 카메라 | 탐지 불가 | 탐지 시도 (렌즈 자체 반사) |

### 4.3 솔직한 한계 고지 (신뢰 차별점)

| 항목 | 기존 앱 | SearCam |
|------|--------|---------|

## 6. 경쟁 전략 (Blue Ocean 접근)

### 6.1 Red Ocean (기존 경쟁)

현재 몰래카메라 탐지 앱 시장은 다음 항목에서 경쟁합니다:
- 탐지 모드 개수 (더 많은 센서)
- AI/ML 기반 이미지 분석 (정확도 불명)
- 유료 기능 범위
- 광고 모델

### 6.2 Blue Ocean 전략 캔버스

**제거 (Eliminate)**:
- 서버 의존 AI 분석 (온라인 필수, 개인정보 리스크)
- 핵심 기능 페이월 (안전 도구의 유료화)
- 결과 블러 처리 (Peek 방식의 강압적 구독 유도)

**감소 (Reduce)**:
- 오탐률 (55~70% → 5~10%)
- 기술 용어 노출 (ARP, mDNS → 숨김)
- 광고 빈도 (스캔 중 광고 없음)

**증가 (Raise)**:
- 결과 투명성 (근거 기반 설명)
- 사용 편의성 (원클릭 통합 스캔)
- 한국 시장 최적화 (언어, 시나리오, 체크리스트)

**창조 (Create)**:
- 교차 검증 엔진 (3개 레이어 종합 분석)
- 플래시 Retroreflection AI 렌즈 감지 (밝은 환경 가능)
- 솔직한 한계 고지 (탐지 불가 범위 안내)
- 육안 체크리스트 (센서 보완)

### 6.3 시장 포지셔닝 맵

```
     정확도 높음
         ▲
         │
         │  ★ SearCam (목표)
         │    (교차 검증 + 렌즈 AI)
         │
    ─────┼──────────────────→ 사용 편의
  복잡   │                    간편
         │
         │  CamX    FindSpy
         │  (다기능  (자동 스캔,
         │   수동)   교차검증X)
         │
         │        Peek
         │        (iOS 전용,
    HCD  │         유료)
    (오탐 │
     많음)│   몰카탐지기
         │   (기본 기능)
         │
     정확도 낮음
```

SearCam의 목표 포지션: **높은 정확도 + 높은 사용 편의성** (우측 상단).
기존 앱은 정확도가 낮거나, 정확도를 위해 복잡한 수동 조작이 필요한 좌측에 분포합니다.
