## 1. 버전 관리 전략

### 1.1 Semantic Versioning (SemVer)

```
MAJOR.MINOR.PATCH

MAJOR: 호환성이 깨지는 변경 (아키텍처 변경, API 변경)
MINOR: 하위 호환되는 기능 추가 (새 탐지 레이어, 새 화면)
PATCH: 하위 호환되는 버그 수정 (오탐 수정, 크래시 수정)
```

### 4.1 개요

| 항목 | 내용 |

| 기간 | M2 W1 ~ M2 W3 (3주) |

### 4.2 베타 테스터 모집

**모집 채널**:
- 몰래카메라 관련 온라인 커뮤니티
- 여행 커뮤니티 (에어비앤비 이용자)

**모집 조건**:
- Android 8.0 이상 기기 소유
- 주 1회 이상 테스트 참여 가능
- 피드백 작성 의지 있는 사용자

**모집 규모**:
- 목표: 100명 모집
- 최소: 50명

### 4.3 베타 피드백 수집 항목

| 카테고리 | 수집 항목 |
|---------|----------|
| 정확도 | 오탐 발생 횟수, 미탐 경험 여부 |
| 사용성 | 스캔 시작까지 소요 시간, 결과 이해도 |
| 성능 | 스캔 소요 시간 체감, 배터리 소모 체감 |

| 환경 | 테스트 장소 유형, 기기 모델 |
| 만족도 | NPS (0~10), 재사용 의향 |

### 4.4 베타 피드백 폼 (앱 내)

```

2. 어떤 스캔을 사용했나요?
   □ Quick Scan □ Full Scan □ 렌즈 찾기 □ IR □ EMF

3. 결과가 정확했나요?
   □ 정확했다 □ 오탐이 있었다 □ 미탐이 있었다 □ 모르겠다

4. 오탐이 있었다면, 무엇이 오탐이었나요?
   [자유 입력]

5. 사용하면서 불편한 점은?
   [자유 입력]

6. 0~10점으로, 이 앱을 추천하시겠습니까? (NPS)
   [슬라이더]

7. 추가 의견
   [자유 입력]
```

```
"없는 것보다 훨씬 나은 1차 스크리닝 도구"입니다.
탐지되지 않았다고 100% 안전을 보장하지 않습니다.

핵심 무료:
모든 탐지 기능은 완전 무료입니다.
프리미엄은 무제한 리포트 저장, PDF 내보내기, 광고 제거만 포함합니다.
```

## 6. 롤아웃 전략 (단계적 출시)

### 6.1 단계별 롤아웃

```
Day 1:   10% 롤아웃 → 모니터링 24시간
Day 2:   크래시/ANR 확인 → 문제 없으면 계속
Day 3:   25% 롤아웃 → 모니터링 48시간
Day 5:   크래시율 < 1%, ANR < 0.5% 확인
Day 5:   50% 롤아웃 → 모니터링 48시간
Day 7:   전체 지표 확인
Day 7:   100% 롤아웃
```

### 6.2 롤아웃 중단 기준

| 지표 | 중단 기준 | 조치 |
|------|----------|------|
| 크래시율 | > 2% | 즉시 중단, 원인 분석 |
| ANR 비율 | > 1% | 즉시 중단, 성능 분석 |
| 1점 리뷰 폭증 | 급증 (일 10건+) | 중단 검토 |
| 핵심 기능 장애 | 스캔 불가 | 즉시 중단, 핫픽스 |

### 6.3 롤아웃 모니터링 체크리스트

각 단계 확장 전 확인:

- [ ] 크래시율 < 1%
- [ ] ANR 비율 < 0.5%
- [ ] 평균 평점 4.0 이상 유지
- [ ] 1점 리뷰 특이 패턴 없음
- [ ] Firebase Crashlytics 새 이슈 없음
- [ ] 스캔 완료율 70% 이상
- [ ] 앱 시작 시간 < 2초

## 7. 핫픽스 프로세스

### 7.1 핫픽스 대상

| 심각도 | 설명 | 대응 시간 |
|--------|------|----------|
| P0 (Critical) | 앱 크래시, 데이터 손실, 보안 취약점 | 24시간 이내 |
| P1 (High) | 핵심 기능 장애 (스캔 불가) | 48시간 이내 |
| P2 (Medium) | 부분 기능 장애 | 다음 정규 릴리스 |
| P3 (Low) | UI 이상, 문구 오류 | 다음 정규 릴리스 |

### 7.2 핫픽스 워크플로우

```
1. 이슈 감지 (Crashlytics / 리뷰 / 내부 보고)
   │
2. 심각도 판정 (P0/P1만 핫픽스)
   │
3. main에서 hotfix/ 브랜치 생성
   │
4. 수정 + 테스트 (Unit + 수동)
   │
5. PR → 코드 리뷰 (1명, 긴급 승인)
   │
6. main 머지 + 태그 (vX.X.X+1)
   │
7. Release 파이프라인 자동 실행
   │
8. Internal → 빠른 QA → Production (100% 즉시)
   │
9. 핫픽스 내용 develop에도 머지
```

### 7.3 핫픽스 버전 규칙

```
현재 버전: 1.0.0
핫픽스 1: 1.0.1
핫픽스 2: 1.0.2

※ 핫픽스는 PATCH 버전만 올림
※ versionCode도 +1 증가
```

```
SearCam v1.0.0 정식 출시

30초 만에 몰래카메라를 점검하세요!

주요 기능:
- Quick Scan: Wi-Fi 네트워크 빠른 스캔 (30초)
- Full Scan: Wi-Fi + 렌즈 감지 + 자기장 3중 점검
- 렌즈 찾기: 플래시로 카메라 렌즈 위치 탐지
- IR 감지: 야간 카메라 적외선 탐지
- 자기장 감지: 전자기기 이상 신호 탐지
- 체크리스트: 숙소/화장실 육안 점검 가이드
- 리포트: 스캔 결과 저장 및 PDF 내보내기

모든 핵심 탐지 기능은 무료입니다.
피드백은 [설정 > 피드백]으로 보내주세요.
```
