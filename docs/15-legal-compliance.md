## 1. 개인정보 보호법 준수

### 1.1 적용 법률

SearCam은 대한민국에서 최초 출시되므로 다음 법률을 우선 준수한다.

| 법률 | 관할 | 핵심 요구사항 |
|------|------|-------------|
| 개인정보 보호법 | 대한민국 | 개인정보 수집 시 동의, 목적 외 이용 금지 |
| 정보통신망 이용촉진 및 정보보호에 관한 법률 | 대한민국 | 통신 관련 개인정보 보호, 네트워크 보안 |
| 위치정보의 보호 및 이용 등에 관한 법률 | 대한민국 | 위치정보 수집/이용 시 사전 동의 |

### 1.2 수집 데이터 목록: "수집 안 함"을 증명

SearCam Phase 1은 개인정보를 수집하지 않는다. 이를 다음과 같이 증명한다.

#### 1.2.1 처리 데이터 분류

| 데이터 | 처리 방식 | 저장 | 외부 전송 | 개인정보 해당 여부 |
|--------|----------|------|----------|-----------------|
| Wi-Fi 기기 MAC 주소 | 메모리에서 분석 | SHA-256 해시만 저장 | 없음 | 해시 처리로 식별 불가 |
| 카메라 영상 프레임 | 메모리에서 실시간 분석 | 저장 안 함 | 없음 | 저장하지 않으므로 해당 없음 |
| 자기장 측정값 | 실시간 표시 | 요약 수치만 리포트 저장 | 없음 | 개인 식별 불가 |
| Wi-Fi SSID | 네트워크 확인용 | 저장 안 함 | 없음 | 저장하지 않으므로 해당 없음 |
| 위치 정보 | Wi-Fi 스캔 API 호출 조건 | 저장 안 함 | 없음 | 수집/저장하지 않음 |
| 스캔 리포트 | 사용자 기기에 로컬 저장 | 로컬 DB (암호화) | 없음 | 사용자 기기 내 관리 |

#### 1.2.2 개인정보 수집 여부 판단

```
개인정보 보호법 제2조 (정의)
"개인정보"란 살아 있는 개인에 관한 정보로서 성명, 주민등록번호 등을 통하여
개인을 알아볼 수 있는 정보를 말한다.

SearCam 판단:
1. MAC 주소 → SHA-256 해시 처리 → 원본 복원 불가 → 비식별 정보
2. 카메라 프레임 → 저장하지 않음 → 수집에 해당하지 않음
3. 자기장 수치 → 개인 식별과 무관
4. 위치 정보 → 시스템 API 조건이므로 사용, 수집/저장하지 않음

결론: 개인정보 보호법상 "개인정보 수집"에 해당하지 않음
```

### 1.3 로컬 처리 원칙

| 원칙 | 구현 |
|------|------|
| 온디바이스 분석 | 모든 탐지 알고리즘이 사용자 기기에서 실행 |
| 서버 미사용 | Phase 1에서 백엔드 서버 없음 |
| 데이터 미전송 | 인터넷 권한은 OUI DB 업데이트 + AdMob 광고에만 사용 |
| 사용자 제어 | 리포트 저장/삭제를 사용자가 직접 관리 |

### 1.4 개인정보 처리방침 필요 여부

```
판단 근거:
- 개인정보를 수집하지 않더라도 Google Play Store 정책상 개인정보 처리방침 URL이 필수
- 앱에서 카메라, 위치 등 민감 권한을 사용하므로 투명성 차원에서 공개 필요
- "수집하지 않는다"는 사실도 처리방침에 명시적으로 기재해야 함

결론: 개인정보 처리방침 작성 및 공개 필요 (수집 안 함을 명시하는 형태)
```

## 2. 정보통신망법 준수

### 2.1 네트워크 스캔의 법적 범위

#### 2.1.1 수동적 스캔 vs 능동적 해킹 분류

| 행위 | 분류 | 합법성 | SearCam 해당 |
|------|------|--------|-------------|
| ARP 테이블 조회 | 수동적 정보 수집 | 합법 (자기 기기 정보 확인) | O |
| mDNS/SSDP 서비스 탐색 | 수동적 수신 (브로드캐스트 응답) | 합법 (표준 프로토콜) | O |
| MAC OUI 매칭 | 공개 DB 조회 | 합법 (IEEE 공개 데이터) | O |
| TCP Connect 포트 확인 | 능동적 연결 시도 | 동일 네트워크 내 합법* | O (제한적) |
| SYN 스캔 / 스텔스 스캔 | 공격적 스캔 | 의도에 따라 불법 가능 | X (사용 안 함) |
| 패킷 스니핑 (Monitor Mode) | 도청 | 불법 (통신비밀보호법) | X (기술적 불가) |
| 트래픽 DPI (Deep Packet Inspection) | 통신 내용 열람 | 불법 | X (기술적 불가) |
| 무선 AP 해킹 / 비밀번호 크래킹 | 해킹 | 불법 (정보통신망법 제48조) | X (기능 없음) |

*동일 네트워크 내 TCP Connect: 사용자가 합법적으로 접속한 네트워크에서 공개 포트를 확인하는 행위는 해킹에 해당하지 않는다. 단, 이를 악용하여 타인의 시스템에 무단 침입하는 것은 불법이다.

#### 2.1.2 포트 스캔의 합법성 분석

```
정보통신망법 제48조 (정보통신망 침해행위 등의 금지)
① 누구든지 정당한 접근권한 없이 또는 허용된 접근권한을 넘어
   정보통신망에 침입하여서는 아니 된다.

SearCam 포트 스캔 분석:

[합법 근거]
1. 사용자가 합법적으로 접속한 네트워크에서 수행
   - 호텔/숙소 Wi-Fi에 정상적으로 연결된 상태
   - 사용자 자신의 네트워크 접속 권한 범위 내

2. TCP Connect 방식 = 정상적인 연결 시도
   - 공개 포트에 대한 연결 시도는 "침입"이 아님
   - 웹 브라우저가 HTTP 포트에 접속하는 것과 동일한 수준

3. 목적의 정당성
   - 불법 몰래카메라 탐지라는 합법적/공익적 목적
   - 개인의 프라이버시 보호를 위한 자구 행위

4. 범위의 최소성
   - 카메라 관련 포트 6개만 확인 (전체 65535포트 스캔이 아님)
   - 연결 확인 후 즉시 종료 (데이터 수집/수정 없음)
   - 같은 서브넷 내에서만 작동

[주의 사항]
- 스캔 결과로 타인의 기기에 "접속"하는 기능은 제공하지 않음
- RTSP 스트리밍을 "시청"하는 기능은 Phase 1에서 제공하지 않음
- 포트가 열려 있다는 사실만 확인 (내용 접근 없음)
```

#### 2.1.3 앱 내 네트워크 스캔 고지 문구

```
[Wi-Fi 스캔 시작 전 안내]

이 기능은 현재 연결된 Wi-Fi 네트워크에서
카메라로 의심되는 기기를 탐색합니다.

• 같은 Wi-Fi에 연결된 기기만 확인합니다
• 기기의 제조사와 서비스 유형을 분석합니다
• 타인의 기기에 접속하거나 내용을 확인하지 않습니다
• 스캔 결과는 기기에만 저장되며 외부로 전송되지 않습니다

[스캔 시작]  [취소]
```

## 3. Google Play Store 정책 준수

### 3.1 카메라 권한 사용 설명

| 정책 요건 | SearCam 대응 |
|----------|-------------|
| 카메라 권한의 명확한 사용 목적 기재 | "카메라 렌즈의 역반사(retroreflection)를 분석하여 숨겨진 카메라를 탐지합니다" |
| 배경(Background)에서의 카메라 사용 금지 | 카메라는 렌즈 찾기/IR 감지 화면에서만 활성화, 백그라운드 사용 없음 |

```
[Play Store 데이터 안전 섹션]

### 4.2 이용약관 초안

```
SearCam 이용약관

제1조 (목적)
본 약관은 SearCam(이하 "앱")의 이용에 관한 조건과 절차를 규정합니다.

제2조 (서비스의 성격)
① 본 앱은 숨겨진 카메라 탐지를 보조하는 1차 스크리닝 도구입니다.
② 본 앱은 전문 탐지 장비를 대체하지 않으며, 100% 탐지를 보장하지 않습니다.
③ 스캔 결과는 참고 정보이며, 최종 판단은 사용자의 책임입니다.

제3조 (탐지 한계)
다음 유형의 카메라는 탐지가 어렵거나 불가능할 수 있습니다:
① Wi-Fi에 연결되지 않은 카메라
② 전원이 꺼진 카메라
③ LTE/5G 독립 통신 카메라
④ SD카드 단독 녹화 카메라
⑤ 렌즈가 물리적으로 가려진 카메라

제4조 (면책)
① 본 앱의 탐지 결과에 의존하여 발생한 직접적·간접적 손해에 대해
   개발자는 책임을 지지 않습니다.
② 스캔 결과에 관계없이, 의심 상황에서는 전문 업체 또는 경찰(112)에
   신고할 것을 권장합니다.
③ 네트워크 스캔은 사용자가 합법적으로 접속한 네트워크 범위 내에서만
   수행되며, 스캔 결과의 악용(타인 기기 무단 접속 등)은 사용자의
   책임입니다.

제5조 (데이터 처리)
① 본 앱은 개인정보를 수집하지 않습니다.
② 모든 탐지 및 분석은 사용자의 기기에서 로컬로 수행됩니다.
③ 스캔 리포트는 사용자 기기에만 저장되며 외부로 전송되지 않습니다.

SearCam 개인정보 처리방침
