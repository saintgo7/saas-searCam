## 1. 체크리스트 설계 원칙

### 1.1 핵심 원칙

SearCam의 센서 기반 탐지는 Wi-Fi 연결 카메라, IR 야간 카메라, 전원이 켜진 소형 카메라를 감지한다. 그러나 **SD카드 녹화 카메라, 배터리+SD 초소형 카메라, 전원 OFF 카메라**는 어떤 센서로도 탐지할 수 없다. 이런 카메라를 발견하는 유일한 방법은 **체계적인 육안 점검**이다.

```
설계 원칙:

  1. 전문 지식 없이도 따라할 수 있는 수준
     ├── 모든 항목에 구체적인 동작 지시 ("이것을 하세요")
     ├── 전문 용어 최소화, 일상 언어 사용
     └── "왜 점검하는지" 짧은 이유 제공

  2. 이미지/일러스트 가이드 포함
     ├── 각 점검 항목에 참고 이미지 또는 일러스트
     ├── "의심스러운 상태" vs "정상 상태" 비교 이미지
     └── 카메라가 위장되는 대표적 형태 사진

  3. 공간 유형별 맞춤화
     ├── 숙소(호텔/에어비앤비): 시선 집중 영역 위주
     ├── 화장실(공중/숙소): 거울, 환풍구, 칸막이
     ├── 탈의실(헬스장/수영장): 옷걸이, 벤치, 로커
     └── 사무실/회의실: 화상회의 장비, 연기감지기

  4. 소요 시간 명시
     ├── 각 체크리스트에 예상 소요 시간 표시
     ├── 항목별 우선순위 (필수/권장/선택)
     └── "빠른 점검" 모드: 상위 5개 항목만
```

### 1.2 체크리스트 공통 구조

모든 체크리스트 항목은 동일한 구조를 따른다:

```
체크리스트 항목 구조:

  [번호] [점검 위치]
    점검 방법: (구체적 동작)
    의심 신호: (이럴 때 의심하세요)
    참고 이미지: (이미지 설명)
    위험도: [높음/중간/낮음]
    우선순위: [필수/권장/선택]
```

## 2. 숙소 체크리스트 (20개 항목)

> **대상**: 호텔, 모텔, 에어비앤비, 게스트하우스
> **예상 소요 시간**: 5~10분
> **빠른 점검 모드**: 항목 1~5 (2분)

### 2.1 시선 방향 점검 (침대/소파를 향한 곳)

**[1] 연기 감지기 (화재 경보기)**

- 점검 위치: 천장에 설치된 연기 감지기. 특히 침대 바로 위 또는 침대를 향한 위치
- 점검 방법:
  - 연기 감지기를 손전등(스마트폰 플래시)으로 비추며 렌즈 구멍이 있는지 확인
  - 연기 감지기 외관을 자세히 관찰. 일반 연기 감지기에는 작은 격자 구멍만 있고, 렌즈처럼 투명한 부분은 없음
  - 가능하면 손으로 살짝 돌려서 쉽게 분리되는지 확인 (진짜 연기 감지기는 단단히 고정됨)
  - 같은 방에 연기 감지기가 2개 이상이면 의심
- 의심 신호: 렌즈처럼 반짝이는 작은 구멍, 플라스틱 돔 안에 검은 점, 기기에서 미세한 LED 불빛, 배선이 벽 안으로 들어가지 않고 외부 노출
- 참고 이미지: 정상 연기 감지기 vs 카메라 위장 연기 감지기 비교. 카메라형은 중앙에 작은 렌즈 구멍이 있고, 측면에 microSD 슬롯이 있을 수 있음
- 위험도: **높음**
- 우선순위: **필수**

**[2] 시계 (벽시계, 탁상시계)**

- 점검 위치: 벽에 걸린 시계, 침대 옆 탁상시계, 특히 침대 방향을 향한 시계
- 점검 방법:
  - 시계 숫자판 주변에 작은 구멍이 있는지 확인
  - 디지털 시계의 경우 디스플레이 옆 검은 부분 주의
  - 시계 뒷면에 microSD 슬롯이나 USB 포트가 있는지 확인
  - 시계가 콘센트에 연결되어 있으면 의심 (일반 벽시계는 건전지)
- 의심 신호: 숫자 사이 또는 테두리에 작은 구멍(1~2mm), 뒷면에 SD카드 슬롯, 콘센트 연결, 시계 브랜드가 불분명
- 참고 이미지: 카메라 내장 시계는 숫자 12 또는 6 위치에 핀홀이 있는 경우가 많음
- 위험도: **높음**
- 우선순위: **필수**

**[3] TV 베젤 / 모니터**

- 점검 위치: TV 화면 테두리(베젤), 특히 상단 중앙
- 점검 방법:
  - TV 상단 베젤에 작은 구멍이나 렌즈가 있는지 확인
  - TV가 꺼진 상태에서 플래시로 베젤을 비추어 반사 확인
  - TV 뒤편에 추가 장치가 부착되어 있는지 확인
- 의심 신호: 베젤에 본래 디자인에 없는 구멍, TV 뒤에 이상한 장치, 전원이 꺼져도 LED가 깜빡임
- 참고 이미지: TV 베젤 상단에 삽입된 핀홀 카메라 예시
- 위험도: **중간**
- 우선순위: **필수**

**[4] 에어컨 리모컨 / 벽부착 리모컨**

- 점검 위치: 에어컨 벽부착 리모컨, 침대 머리맡 리모컨
- 점검 방법:
  - 리모컨의 IR 송신부(투명 부분)를 자세히 관찰
  - 리모컨에 본래 기능과 무관한 구멍이 있는지 확인
  - 벽부착 리모컨이 콘센트에 별도 연결되어 있으면 주의
- 의심 신호: IR 송신부 외에 추가 투명 부분, 리모컨이 유난히 두꺼움, 뒷면에 SD 슬롯
- 참고 이미지: 에어컨 리모컨 외형의 위장 카메라 제품 예시
- 위험도: **중간**
- 우선순위: **권장**

**[5] USB 충전기 / 전원 어댑터**

- 점검 위치: 콘센트에 꽂힌 USB 충전기, 벽면 충전기, 멀티탭
- 점검 방법:
  - 충전기 전면에 USB 포트 외 작은 구멍이 있는지 확인
  - 충전기가 유난히 크거나, 일반적인 브랜드가 아닌 경우 주의
  - 충전기에 스마트폰을 연결해보고 실제 충전이 되는지 확인
  - 충전 포트 외에 핀홀이 있으면 의심
- 의심 신호: USB 포트 옆 또는 위에 1mm 구멍, 비정상적 크기, 알 수 없는 브랜드, LED 표시등이 있지만 충전과 무관하게 점등
- 참고 이미지: 정상 USB 충전기 vs 카메라 내장 충전기 비교. 카메라형은 전면 상단에 핀홀이 있음
- 위험도: **높음**
- 우선순위: **필수**

### 2.2 벽면/천장 점검

**[6] 액자 / 그림**

- 점검 위치: 벽에 걸린 액자, 특히 침대/소파를 향한 위치
- 점검 방법:
  - 액자 프레임에 작은 구멍이 있는지 확인 (특히 모서리)
  - 액자를 살짝 들어서 뒤에 장치가 부착되어 있는지 확인
  - 액자 유리 뒤에 렌즈처럼 보이는 부분이 있는지 확인
- 의심 신호: 프레임에 핀홀, 뒤에 전선, 벽에 단단히 고정(분리 불가)
- 참고 이미지: 액자 프레임 모서리에 핀홀 카메라가 삽입된 예시
- 위험도: **중간**
- 우선순위: **권장**

**[7] 나사 / 못 / 볼트**

- 점검 위치: 벽면의 나사, 특히 위치가 기능적으로 불필요해 보이는 나사
- 점검 방법:
  - 벽의 나사 머리를 가까이에서 관찰
  - 나사 머리 중앙에 렌즈처럼 보이는 부분이 있는지 확인
  - 나사가 비대칭적으로 설치되었거나, 주변 나사와 다른 형태
  - 플래시로 비추어 나사 머리에서 렌즈 반사가 있는지 확인
- 의심 신호: 나사 머리 중앙이 투명하거나 반짝임, 기능 없는 위치의 나사, 주변 나사와 크기/색상이 다름
- 참고 이미지: 나사 모양 핀홀 카메라. 일반 나사와 외형은 거의 동일하나 머리 중앙에 미세 렌즈 존재
- 위험도: **높음**
- 우선순위: **필수**

**[8] 환풍구 / 에어컨 통풍구**

- 점검 위치: 벽면/천장의 환풍구, 에어컨 통풍구
- 점검 방법:
  - 환풍구 격자 사이를 플래시로 비추며 내부 관찰
  - 격자 사이에 렌즈 같은 반짝이는 점이 있는지 확인
  - 환풍구에서 바람이 실제로 나오는지 확인 (위장 환풍구 여부)
- 의심 신호: 격자 사이에 반짝이는 점, 바람이 나오지 않는 환풍구, 새로 설치된 듯한 깨끗한 환풍구
- 참고 이미지: 환풍구 격자 뒤에 숨겨진 카메라 예시
- 위험도: **중간**
- 우선순위: **권장**

**[9] 스프링클러 (소방 시설)**

- 점검 위치: 천장의 스프링클러 헤드
- 점검 방법:
  - 스프링클러 헤드를 플래시로 비추어 내부에 렌즈가 있는지 확인
  - 정상 스프링클러에는 유리 벌브(빨간색/오렌지색)가 있음
  - 유리 벌브 대신 검은색 렌즈가 보이면 의심
- 의심 신호: 유리 벌브가 없고 검은 렌즈가 보임, 같은 방에 스프링클러가 2개 이상, 위치가 침대를 향함
- 참고 이미지: 정상 스프링클러 헤드(유리 벌브 포함) vs 위장 카메라
- 위험도: **중간**
- 우선순위: **권장**

**[10] 조명 기구 / 전등**

- 점검 위치: 천장 조명, 벽면 간접조명, 독서등
- 점검 방법:
  - 조명 기구 내부에 렌즈가 있는지 확인 (전원 끄고 관찰)
  - 조명 커버를 통해 내부 부품 관찰
  - 조명 장치에서 침대 방향으로 작은 구멍이 있는지 확인
- 의심 신호: 조명 내부에 본래 없어야 할 부품, 렌즈처럼 보이는 부분, LED 외 별도 전자 부품
- 참고 이미지: 조명 기구 내부에 설치된 소형 카메라 예시
- 위험도: **중간**
- 우선순위: **선택**

### 2.3 욕실 점검

**[11] 거울 (양면 거울 테스트)**

- 점검 위치: 욕실 거울, 드레스룸 거울, 벽면 거울 전체
- 점검 방법: (아래 섹션 3에서 상세 설명)
  - 손톱 테스트: 손톱을 거울에 대고 반사 이미지와의 간격 확인
  - 조명 테스트: 방을 어둡게 하고 거울 뒤에 플래시를 비춤
  - 두드리기 테스트: 거울을 가볍게 두드려 빈 공간 소리 확인
- 의심 신호: 손톱과 반사 이미지 사이 간격 0 (양면 거울), 거울 뒤에서 빛이 투과됨, 빈 공간 소리
- 참고 이미지: 정상 거울(간격 있음) vs 양면 거울(간격 없음) 비교 사진
- 위험도: **높음**
- 우선순위: **필수**

**[12] 샤워기 홀더 / 샤워 헤드**

- 점검 위치: 샤워기 홀더(벽 부착형), 샤워 헤드 본체
- 점검 방법:
  - 샤워기 홀더의 관절 부분에 렌즈 구멍 확인
  - 샤워 헤드의 물 나오는 면 외에 다른 구멍 확인
  - 홀더가 특정 방향(사용자 몸쪽)으로 고정되어 있는지 확인
- 의심 신호: 홀더에 핀홀, 샤워 헤드에 물 구멍 외 추가 구멍, 비정상적 각도 고정
- 참고 이미지: 샤워기 홀더 위장 카메라 예시
- 위험도: **높음**
- 우선순위: **권장**

**[13] 비누/샴푸통 거치대**

- 점검 위치: 벽부착 비누통, 샴푸 디스펜서, 치약 거치대
- 점검 방법:
  - 디스펜서 상단/전면에 작은 구멍이 있는지 확인
  - 벽에 고정된 디스펜서는 분리하여 뒤쪽 확인 (가능한 경우)
  - 비정상적으로 새로워 보이는 디스펜서 주의
- 의심 신호: 디스펜서에 기능과 무관한 구멍, 내용물이 들어있지 않은 디스펜서, 벽에 단단히 고정(분리 불가)
- 참고 이미지: 비누 디스펜서 위장 카메라 제품 예시
- 위험도: **중간**
- 우선순위: **권장**

**[14] 휴지 거치대 / 휴지 케이스**

- 점검 위치: 벽부착 휴지 거치대, 특히 나사 부분
- 점검 방법:
  - 휴지 거치대의 나사 머리를 자세히 확인
  - 거치대 상단이나 측면에 핀홀이 있는지 확인
  - 거치대가 변기/샤워 방향을 향해 설치되었는지 확인
- 의심 신호: 나사 머리에 렌즈, 거치대에 기능 외 구멍, 비정상적 위치 설치
- 참고 이미지: 휴지 거치대 나사 부분에 설치된 핀홀 카메라
- 위험도: **중간**
- 우선순위: **권장**

**[15] 콘센트 / 멀티탭**

- 점검 위치: 욕실 콘센트, 드라이기 콘센트 주변
- 점검 방법:
  - 콘센트 커버에 작은 구멍이 있는지 확인
  - 멀티탭에 USB 포트 외 추가 구멍 확인
  - 콘센트 커버가 교체된 흔적(색상 차이, 스크래치)이 있는지 확인
- 의심 신호: 콘센트에 본래 없는 구멍, 멀티탭에 핀홀, 커버 교체 흔적
- 참고 이미지: 콘센트 커버 위장 카메라 예시
- 위험도: **중간**
- 우선순위: **선택**

### 2.4 기타 점검

**[16] 멀티탭 / 연장 코드**

- 점검 위치: 방 곳곳의 멀티탭, 책상 아래 연장 코드
- 점검 방법:
  - 멀티탭에 USB 포트 옆 핀홀 확인
  - 멀티탭이 침대/소파를 향해 놓여 있는지 확인
  - 일반적이지 않은 브랜드의 멀티탭 주의
- 의심 신호: USB 포트 옆 핀홀, 침대 방향 배치, 불명 브랜드, LED가 점등되지만 기능 불분명
- 참고 이미지: 멀티탭 위장 카메라 (USB 포트 옆에 핀홀 렌즈)
- 위험도: **높음**
- 우선순위: **권장**

**[17] 옷걸이 고리 / 코트훅**

- 점검 위치: 벽면 옷걸이 고리, 뒤쪽 훅
- 점검 방법:
  - 옷걸이 고리 상단 볼트/나사 부분에 렌즈 확인
  - 고리가 침대 방향을 향하는지 확인
  - 고리 본체가 비정상적으로 두껍다면 내부 확인
- 의심 신호: 볼트 중앙에 렌즈, 비정상적 두께, 침대 향한 방향
- 참고 이미지: 옷걸이 고리 위장 카메라. 볼트 부분에 초소형 렌즈 내장
- 위험도: **높음**
- 우선순위: **권장**

**[18] 인형 / 장식품**

- 점검 위치: 침대 주변 인형, 장식품, 꽃병
- 점검 방법:
  - 인형의 눈 부분을 자세히 확인 (렌즈 여부)
  - 장식품에 작은 구멍이 있는지 확인
  - 무게가 비정상적(전자 부품 내장)인지 확인
- 의심 신호: 인형 눈이 렌즈처럼 반짝임, 장식품에 핀홀, 비정상적 무게
- 참고 이미지: 인형 눈에 카메라가 내장된 예시
- 위험도: **낮음**
- 우선순위: **선택**

**[19] 공유기 / 네트워크 기기**

- 점검 위치: 방 내 Wi-Fi 공유기, 셋톱박스
- 점검 방법:
  - 공유기에 본래 디자인에 없는 구멍이 있는지 확인
  - 공유기가 침대 방향으로 배치되어 있는지 확인
  - 공유기 브랜드를 확인하고 알려진 제조사인지 판별
- 의심 신호: 안테나 외에 추가 돌출부, 불명 브랜드, 침대 향한 배치, 공유기 역할을 하지 않는 기기
- 참고 이미지: 공유기 위장 카메라 (실제 Wi-Fi 기능 + 카메라 내장)
- 위험도: **중간**
- 우선순위: **선택**

**[20] 천장 타일 / 틈새**

- 점검 위치: 천장 타일의 들뜬 부분, 벽과 천장 사이 틈새
- 점검 방법:
  - 천장 타일에 작은 구멍이나 들뜬 부분 확인
  - 틈새에 플래시를 비추어 렌즈 반사 확인
  - 최근 수리 흔적(색상 차이)이 있는 타일 주의
- 의심 신호: 타일에 핀홀, 들뜬 타일, 최근 수리 흔적, 타일 사이에 검은 점
- 참고 이미지: 천장 타일 사이 틈새에 설치된 소형 카메라
- 위험도: **낮음**
- 우선순위: **선택**

## 3. 화장실 체크리스트 (15개 항목)

> **빠른 점검 모드**: 항목 1~5 (1분)

- 점검 방법: 격자 사이를 플래시로 비추어 내부 확인

- 점검 위치: 칸막이 벽면 휴지 거치대
- 점검 방법: 거치대 상단/나사 부분 핀홀 확인

## 4. 양면 거울 테스트 상세

### 4.1 손톱 테스트 (가장 간단)

```
방법:
  1. 손톱 끝을 거울 표면에 직각으로 대세요
  2. 손톱과 거울 속 반사 이미지 사이의 간격을 확인하세요

판정:
  ├── 간격이 있음 (2~5mm): 정상 거울
  │   일반 거울은 유리 두께만큼 간격이 생김
  │   (유리 앞면 → 은박 코팅 → 반사)
  │
  └── 간격이 0 (손톱이 반사 이미지에 닿음): 양면 거울 의심
      양면 거울은 은박이 유리 앞면에 코팅됨
      뒤에서 볼 때 투명, 앞에서 볼 때 거울

주의사항:
  ├── 일부 고급 거울(무테 거울)도 간격이 작을 수 있음
  ├── 확실하지 않으면 조명 테스트를 병행
  └── 아크릴 거울은 손톱 테스트가 부정확할 수 있음
```

### 4.2 조명 테스트 (가장 확실)

```
방법:
  1. 방의 모든 조명을 끄세요 (가능한 한 어둡게)
  2. 스마트폰 플래시를 거울 표면에 직접 대고 비추세요
  3. 거울 뒤쪽에 빛이 투과되는지 확인하세요
  4. 손으로 플래시 주변을 감싸 외부 빛을 차단하면 더 정확

판정:
  ├── 뒤가 안 보임: 정상 거울
  │   일반 거울은 빛을 완전히 반사하므로 뒤가 보이지 않음
  │
  └── 뒤가 보임 (어둡게나마 뒤쪽 공간이 보임): 양면 거울 확정
      양면 거울은 빛의 일부가 투과되므로
      어두운 환경에서 플래시를 비추면 뒤쪽이 보임
```

### 4.3 두드리기 테스트 (보조)

```
방법:
  1. 거울 표면을 손가락 관절로 가볍게 두드리세요
  2. 소리를 들으세요

판정:
  ├── 둔탁한 소리: 정상 거울 (벽에 직접 부착)
  └── 빈 공간 울림: 뒤에 공간이 있음 → 추가 확인 필요
      (반드시 양면 거울은 아님, 매입형일 수 있음)
```

## 7. 체크리스트 완료 후 조치 안내

### 7.1 의심 물체 발견 시

```
앱 내 안내:

  의심스러운 물체를 발견하셨나요?

  1. 사진을 찍어두세요
     ├── 의심 물체의 전체 모습
     ├── 핀홀/렌즈 부분 클로즈업
     └── 물체의 위치 (방 전체 사진)

  2. 직접 만지거나 분해하지 마세요
     ├── 증거 훼손 방지
     └── 감전 위험

  3. 관리자에게 문의하세요
     ├── 숙소: 프론트 데스크 / 호스트에게 문의
     ├── 공공시설: 관리 사무소에 문의
     └── 설명을 요구하고, 납득할 수 없으면 다음 단계

  4. 경찰에 신고하세요
     ├── [112 전화 걸기] 버튼 (원터치)
     ├── 위치: GPS 좌표 자동 포함
     ├── 스캔 결과 공유 가능 (PDF)
     └── 상담 전 사진+스캔 리포트 준비
```

### 7.2 이상 없음 (체크리스트 완료)

```

  결과: 의심 물체 발견되지 않음

  참고 안내:
  ├── 육안 점검으로 모든 카메라를 발견할 수는 없습니다
  ├── 추가로 SearCam의 Full Scan을 실행해보세요
  └── 불안하시면 전문 탐지 업체에 의뢰하세요

  [리포트 저장]  [Full Scan 하기]  [완료]
```
