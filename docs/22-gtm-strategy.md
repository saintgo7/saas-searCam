## 1. 시장 분석

### 1.1 TAM (Total Addressable Market)

전세계 스마트폰 사용자 중 숙소(호텔, 에어비앤비, 모텔 등)를 이용하는 모든 사용자가 잠재 시장이다.

| 항목 | 수치 | 출처/근거 |
|------|------|----------|
| 전세계 스마트폰 사용자 | 약 50억 명 | Statista 2025 |
| 연간 숙박 여행자 수 | 약 14억 명 | UNWTO 2025 |
| 프라이버시 우려 응답율 | 약 68% | Kaspersky 글로벌 서베이 |
| TAM 추정 | **약 9.5억 명** | 14억 x 68% |
| TAM 매출 규모 (연) | **약 $2.85B** | 9.5억 x 연 $3 ARPU |

### 1.2 SAM (Serviceable Addressable Market)

한국 내 숙소 이용자 및 1인 가구 거주자를 SAM으로 정의한다.

| 항목 | 수치 | 근거 |
|------|------|------|
| 한국 스마트폰 보급률 | 약 97% (약 5,000만 명) | KISDI 2025 |
| 연간 숙박 이용자 | 약 3,800만 건 (중복 포함) | 한국관광공사 |
| 1인 가구 수 | 약 750만 가구 | 통계청 2025 |
| 몰카 불안 경험자 비율 | 약 72% (여성), 35% (남성) | 여성가족부 실태조사 |
| 공중화장실/탈의실 이용 빈도 | 월 4회 이상 (성인 기준) | 추정 |
| SAM 추정 사용자 | **약 1,500만 명** | 숙소 이용자 + 1인가구 중 불안 경험자 |
| SAM 매출 규모 (연) | **약 ₩522억** | 1,500만 x ₩3,480 ARPU |

### 1.4 시장 트렌드

1. **몰래카메라 사회 문제화**: 한국에서 연간 5,000건 이상 불법촬영 신고 (경찰청 통계). 사회적 관심 지속 상승.
2. **1인 가구 증가**: 750만 가구 돌파. 혼자 숙소/원룸 이동 시 안전 확인 니즈 증가.
3. **여행 회복**: 코로나 이후 여행 수요 폭발적 회복. 에어비앤비/소규모 숙소 이용 증가.
4. **프라이버시 의식 강화**: Z세대 중심으로 디지털 프라이버시 관심 급증.
5. **DIY 안전 문화**: 전문 업체 의뢰보다 앱/도구로 자가 점검하는 경향 증가.

## 2. 마케팅 채널 전략

### 2.1 ASO (App Store Optimization)

Play Store에서 유기 검색 트래픽이 전체 설치의 60% 이상을 차지한다. ASO는 가장 중요한 채널이다.

#### 핵심 키워드 (한국어)

| 우선순위 | 키워드 | 월간 검색량 (추정) | 경쟁도 |
|---------|--------|------------------|--------|
| 1 | 몰카 탐지 | 22,000 | 중 |
| 2 | 몰래카메라 탐지기 | 18,000 | 중 |
| 3 | 숨은카메라 찾기 | 12,000 | 저 |
| 4 | 카메라 탐지 앱 | 8,000 | 중 |
| 5 | 몰카 검사 | 6,500 | 저 |
| 6 | 숙소 몰카 | 5,000 | 저 |
| 7 | 원룸 몰카 탐지 | 3,000 | 저 |
| 8 | 화장실 몰카 | 4,500 | 저 |
| 9 | 에어비앤비 카메라 | 2,500 | 저 |
| 10 | 렌즈 탐지기 | 1,500 | 저 |

#### 앱 제목 최적화

```
SearCam - 몰카 탐지기 | 숨은카메라 찾기
```

제목에 핵심 키워드 2개를 자연스럽게 포함한다.

#### 스크린샷 전략

| 순서 | 내용 | 메시지 |
|------|------|--------|
| 1 | Quick Scan 결과 화면 (안전) | "30초면 충분합니다" |
| 2 | 위험 감지 결과 (경고 상태) | "Wi-Fi 카메라를 찾아냅니다" |
| 3 | 렌즈 감지 화면 (플래시 모드) | "플래시로 렌즈를 찾습니다" |
| 4 | 교차 검증 결과 상세 | "왜 위험한지 근거를 알려드립니다" |
| 5 | 체크리스트 화면 | "육안 점검 가이드 포함" |
| 6 | PDF 리포트 | "결과를 리포트로 저장하세요" |

#### A/B 테스트 항목

- 앱 아이콘: 방패형 vs 카메라렌즈형 vs 돋보기형
- 스크린샷 첫 장: 안전 결과 vs 위험 감지 결과
- 앱 설명 첫 줄: 공포 소구 vs 안심 소구
- 프리미엄 CTA 위치: 결과 화면 vs 리포트 화면

## 5. 바이럴 전략

### 5.1 인앱 공유 기능

| 공유 트리거 | 공유 콘텐츠 | 채널 |
|-----------|-----------|------|
| 스캔 완료 후 | "이 숙소는 SearCam으로 안전 점검 완료!" 이미지 카드 | 카카오톡, 인스타 스토리 |
| 위험 감지 시 | "주의! 이 장소에서 의심 신호 발견" (위치 미포함) | 카카오톡 |
| 리포트 PDF | 스캔 리포트 PDF 공유 | 이메일, 메신저 |
| 체크리스트 완료 후 | "숙소 안전 점검 체크리스트 완료!" 배지 | SNS 전체 |

### 5.2 바이럴 계수 (K-factor) 목표

```
K = 초대 수 x 전환율
목표: K = 0.3 (1명이 0.3명을 추가 유입)
- 공유율: 사용자의 10%가 1회 이상 공유
- 공유당 전환율: 공유를 본 사람의 30%가 설치
- K = 0.10 x 3 x 0.30 = 0.09 (현실적 추정)
```

K-factor 0.09는 공격적이지 않지만, 유기 채널과 결합하면 자연 성장에 기여한다.

### 5.3 바이럴 부스터

1. **시즌 이벤트**: 여름 성수기, 명절 연휴 전 "안전 여행 캠페인"
2. **뉴스 트리거**: 몰카 관련 뉴스 발생 시 SNS 신속 대응 콘텐츠
3. **공포 콘텐츠 리밸런스**: 공포로 관심을 끌되, 해결책(앱)으로 마무리
4. **사용자 스토리**: 실제 탐지 사례를 익명으로 공유 (동의 필수)
